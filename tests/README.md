MINIMUM REQUIRMENT OF CODECEPTIONS
=======================================
  1. Codeception 1.6.4
  2. PHP 5.4

 ---------------------
 Installation
 --------------------

  1. Install Codeception via Composer
  2. Add codeception/phpbuiltinserver: "*" to your composer.json
  3. Run composer install
